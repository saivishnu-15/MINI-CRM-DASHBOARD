# 📊 Mini CRM Dashboard

A full-stack Customer Relationship Management system to manage clients, track tasks, and update statuses.

---

## 🚀 Features

- 👤 Add / Edit / Delete clients
- 📌 Task management system
- 📊 Status tracking (Active / Pending / Completed)
- 🔍 Dashboard view of all records
- 💾 MongoDB database storage
- ⚡ Simple and responsive UI

---

## 🛠️ Tech Stack

- Frontend: React (Vite)
- Backend: Node.js, Express.js
- Database: MongoDB
- Styling: CSS
